Welcome to BubbleMi!

BubbleMi is a crypto data visualization site displaying coin data as user interactive bubbles. This was made possible with Coin Gecko's API and 3 sets of D3 libaries. Options at the top allow for filtering by top coins by market capitalization and coins that are either in the GREEN (up 24hr) OR in the RED (down 24hr).
